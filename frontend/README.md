# Autéllo Frontend

Современный фронтенд для системы заявок Autéllo, построенный на React + Vite.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик и dev-сервер
- **Tailwind CSS** - Стилизация
- **Framer Motion** - Анимации
- **Axios** - HTTP клиент

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```
VITE_API_URL=http://localhost:8000
```

## Особенности

- Красивый современный дизайн с эффектом стекла (glassmorphism)
- Анимированный фон с автомобильной тематикой
- Полная форма заявки со всеми полями
- Автоматическое отслеживание метрик поведения пользователя
- Адаптивный дизайн для всех устройств
- Использование Google Fonts (Inter, Playfair Display)
